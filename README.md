# ☕ Java Programming 

This repository contains a collection of Java programs covering basic to intermediate concepts. It is designed for learning, practicing, and strengthening core Java fundamentals.

---

## 📚 Topics Covered

### 🔹 Basics
- Sum of numbers
- Average calculation
- Multiplication tables
- Number patterns
- Leap year check
- Unit conversions (km to miles, Celsius to Fahrenheit)

### 🔹 Control Statements
- If-else conditions
- Loops (for, while)
- Recursion (factorial, Fibonacci, patterns)

### 🔹 Arrays
- Array traversal
- Reverse array
- Sum of elements
- Sorting check
- Searching (min, max, element presence)

### 🔹 Strings
- String manipulation
- Replace characters
- Lowercase conversion
- Detect spaces

### 🔹 Functions
- Mathematical operations
- Recursion-based functions
- Utility programs

### 🔹 Object-Oriented Programming (OOP)
- Classes & Objects
- Constructors
- Method Overloading
- Inheritance
- Polymorphism
- Abstraction
- Interfaces

### 🔹 Exception Handling
- Try-catch examples
- Runtime errors
- Custom handling

### 🔹 Multithreading
- Thread creation
- Priority handling
- Continuous execution

### 🔹 Mini Programs
- Rock Paper Scissors 🎮
- Guessing Game 🎯
- Online Library System 📚
- Encryption/Decryption 🔐

---

## 📂 File Structure

Each `.java` file represents an individual program demonstrating a specific concept.

Example:
- `factorial_recursion.java` → Factorial using recursion  
- `array_traversal_reverse.java` → Reverse an array  
- `Animal_inheritance.java` → Inheritance example  
- `interface_tv_remote.java` → Interface implementation
